# Todo App

This is a simple Todo application designed to help you organize tasks efficiently.

## Features

- Add tasks to your list
- Mark tasks as completed
- Delete tasks from the list
- Edit tasks 
- Responsive design for various devices

## Technologies Used

- React JS
- Redux Toolkit
- Tailwind CSS
